AI & ML Project

Project Title

AI Adaptive Assistance (Athena)

This project was developed as part of the Introduction to AI & ML course in the first semester.

Project Description

Athena is an AI-powered voice assistant designed to help beginners learn how to use computers easily. The assistant accepts voice commands, explains the steps visually, and performs tasks automatically so users can learn by watching.

The project focuses on improving digital literacy, especially for users who are not comfortable with typing or English-based interfaces.


---

Key Features

Voice command interaction

Speech-to-Text processing

AI-based question answering

Desktop automation

Opening applications automatically

Creating and deleting files

Web searching and opening websites

Text-to-speech feedback
